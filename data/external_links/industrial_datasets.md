# External industrial datasets for later exercises / project work

This first exercise is self-contained. For later exercises and projects, the following public datasets are recommended.

## Kolektor Surface-Defect Dataset (KolektorSDD / KSDD)

Official page: https://www.vicos.si/resources/kolektorsdd/

Use case: industrial surface defect detection / segmentation.

License: Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International License (CC BY-NC-SA 4.0), according to the official dataset page.

Citation:
Tabernik, D., Šela, S., Skvarč, J., & Skočaj, D. (2020).
Segmentation-based deep-learning approach for surface-defect detection.
Journal of Intelligent Manufacturing, 31, 759–776.

## Kolektor Surface-Defect Dataset 2 (KolektorSDD2 / KSDD2)

Official page: https://www.vicos.si/resources/kolektorsdd2/

Use case: surface-defect detection, segmentation, weak / mixed supervision.

Citation:
Božič, J., Tabernik, D., & Skočaj, D. (2021).
Mixed supervision for surface-defect detection: from weakly to fully supervised learning.
Computers in Industry, 129, 103459.
https://doi.org/10.1016/j.compind.2021.103459

## MVTec Anomaly Detection Dataset (MVTec AD)

Official page: https://www.mvtec.com/research-teaching/datasets/mvtec-ad

Use case: industrial visual anomaly detection and anomaly localization.

Citation:
Bergmann, P. et al. (2021).
The MVTec Anomaly Detection Dataset: A Comprehensive Real-World Dataset for Unsupervised Anomaly Detection.
International Journal of Computer Vision.
https://doi.org/10.1007/s11263-020-01400-4

## MVTec AD 2

Official page: https://www.mvtec.com/research-teaching/datasets/mvtec-ad-2

Use case: more challenging industrial anomaly detection scenarios, including lighting changes and difficult use cases.

## AITEX Fabric Image Database

Official page: https://www.aitex.es/afid/

Use case: textile defect detection and texture inspection.


## Anomalib

Documentation: https://anomalib.readthedocs.io/

Use case: later anomaly-detection baselines and project work.

Citation:
Akcay, S. et al. (2022).
Anomalib: A Deep Learning Library for Anomaly Detection.
IEEE International Conference on Image Processing (ICIP).
https://doi.org/10.1109/ICIP46576.2022.9897283
