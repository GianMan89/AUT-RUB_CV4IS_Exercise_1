# Data used in Exercise 1

## `synthetic/`

Generated images for controlled demonstrations:
- `checkerboard_highres.png`
- `sine_grating_highfreq.png`
- `brick_wall_synthetic.png`

These are used for sampling and aliasing tasks.

## `industrial_like/`

Small generated industrial-style surface images and masks:
- nominal surface examples
- bright scratch
- dark scratch
- spot defect
- edge chip

These images are not a substitute for real industrial benchmark data, but they are useful for a first Binder exercise because they are small, stable, and self-contained.
